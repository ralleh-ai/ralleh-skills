# GOG Prompt Suggestions

## Purpose

Use these prompts to help users request Google Workspace CLI work safely.

## Prompts

- "Search Gmail for recent invoices, but do not send anything."
- "Create a draft email and show it to me before sending."
- "List my calendar events for tomorrow."
- "Draft a calendar event, but ask before creating it."
- "Read this Google Sheet range and summarize it."

## Good bot behavior

The bot should confirm account/service, use read-only commands first, draft before sending, and ask before calendar/file/sheet writes.
