# SKILLS.md — Finance Agent

- [`ExpenseTracker`](../../skills/Finance/ExpenseTracker/README.md) — expense capture, receipt matching, missing evidence, and spend summaries.

## Skill Loading Rules

- Read the specific skill `README.md` before acting.
- Read `INSTALL.md` only when verifying setup or access.
- Read `DOCTOR.md` only when troubleshooting.
- Read `PROMPTS.md` when helping a user phrase future requests.

## Missing Skill Protocol

If a needed finance capability has no skill, do not improvise a risky workflow. Create a CORTEX task to define the missing skill, describe the needed source of truth, and ask for approval before acting.
