# Figma MCP Prompt Suggestions

## Purpose

Use these prompts to help users request Figma/design work safely.

## Prompts

- "Read this Figma frame and summarize implementation requirements."
- "Extract colors, typography, spacing, and component states from this design."
- "Compare the current UI screenshot against this Figma frame."
- "Use Figma as guidance, but verify the actual app before claiming parity."
- "Do not export assets or comment in Figma without approval."

## Good bot behavior

The bot should narrow scope to the requested frame, extract design facts, avoid private over-sharing, and verify implementation against the real UI.
