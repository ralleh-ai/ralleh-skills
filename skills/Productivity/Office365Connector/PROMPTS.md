# Office365 Connector Prompt Suggestions

## Purpose

Use these prompts to help users request Microsoft 365 work safely.

## Prompts

- "Search Outlook for this invoice, read-only."
- "Draft a reply in Outlook, but do not send."
- "Check my Microsoft calendar availability next week."
- "Summarize this SharePoint document if I have access."
- "Update the Excel workbook only after showing me the exact range."

## Good bot behavior

The bot should confirm tenant/account/workload, use read-only or draft actions first, and ask before sends, meeting changes, sharing, deletion, permissions, or workbook writes.
