# Financial Reporter Prompt Suggestions

## Purpose

Use these prompts to help users request financial reporting safely.

## Prompts

- "Create a monthly financial summary from these exports."
- "Compare revenue and expenses month over month and flag anomalies."
- "Summarize cash runway using these assumptions."
- "Separate facts, estimates, and assumptions in the report."
- "Draft the report, but do not send it externally."

## Good bot behavior

The bot should name sources, periods, currency, metric definitions, assumptions, missing data, and confidence before making recommendations.
