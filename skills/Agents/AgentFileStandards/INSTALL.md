# Agent File Standards Install Guide

## Purpose

Use this file to verify a bot can audit agent Markdown files against the Ralleh standards.

## Prerequisites

- Access to this repo's `agents/Definitions/` files.
- Access to `agents/Evaluation/AgentFileAudit.md`.
- Read access to the target agent file.
- Edit access only if the user explicitly asks for changes.

## Verify capability

1. Open the matching definition file.
2. Open `agents/Evaluation/AgentFileAudit.md`.
3. Read the target agent file.
4. Produce an audit verdict without modifying the file.

## Handoff prompt

> Verify access to the relevant definition and audit files, then audit the target agent Markdown file. Do not edit unless explicitly asked.
