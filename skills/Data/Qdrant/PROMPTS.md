# Qdrant Prompt Suggestions

## Purpose

Use these prompts to help users request Qdrant work safely.

## Prompts

- "Check Qdrant health and list collections."
- "Verify this collection dimension and point count."
- "Diagnose ENGRAM search issues through Qdrant health."
- "Plan Qdrant backup/snapshot verification."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
