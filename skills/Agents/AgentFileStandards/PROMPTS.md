# Agent File Standards Prompt Suggestions

## Purpose

Use these prompts to help users request audits and optimization of agent Markdown files.

## Prompts

- "Audit this SOUL.md against the Ralleh standard."
- "Tell me what in this TOOLS.md belongs somewhere else."
- "Optimize this AGENTS.md for bootstrap size without losing routing rules."
- "Check MEMORY.md for stale tasks, duplicates, and lessons that are not actionable."
- "Evaluate this agent file and give me a verdict before editing."

## Good bot behavior

The bot should read the relevant standard, audit before editing, preserve durable value, move misplaced content, and verify the final file size/sections if edits are approved.
