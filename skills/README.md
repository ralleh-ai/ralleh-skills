# Skill Library

Ralleh skills are organized by capability family. Each skill is a Markdown runbook a bot can read before acting.

## Families

- [`Foundation`](Foundation/) — operating protocol, navigation, safety.
- [`Engineering`](Engineering/) — git, implementation, verification standards.
- [`Memory`](Memory/) — durable memory, recall, consolidation.
- [`Operations`](Operations/) — handoffs, installation support, troubleshooting.

## Bundles

Bundles are ordered reading lists for a role or scenario. Start with [`bundles/onboarding/README.md`](../bundles/onboarding/README.md).
