# LinkedIn Manager Prompt Suggestions

## Purpose

Use these prompts to help users request LinkedIn work safely.

## Prompts

- "Draft a LinkedIn post about this, but do not publish."
- "Review this post for clarity, credibility, and tone."
- "Create three variants for a company page update."
- "Check the page if logged in, but stop before posting."
- "Prepare an outreach message, but do not send it."

## Good bot behavior

The bot should draft and review freely, verify facts, respect brand voice, and ask before any public/direct LinkedIn action.
