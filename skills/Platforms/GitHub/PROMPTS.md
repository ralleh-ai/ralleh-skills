# GitHub Prompt Suggestions

## Purpose

Use these prompts to help users request GitHub work safely.

## Prompts

- "Check the open PRs and summarize blockers."
- "Inspect the failed GitHub Actions run and tell me the root cause."
- "Create a PR for this branch, but show me the body first."
- "Review this issue and suggest the next implementation step."
- "Do not comment publicly until I approve."
- "Verify the repo/branch before pushing."

## Good bot behavior

The bot should verify repo/account/branch, use structured output, ask before public writes, and cite CI logs or diffs when making claims.
