# Postgres Prompt Suggestions

## Purpose

Use these prompts to help users request Postgres work safely.

## Prompts

- "Check Postgres health and list databases."
- "Plan a safe migration with backup and validation."
- "Verify Postgres is not publicly exposed."
- "Diagnose connection failures without printing secrets."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
