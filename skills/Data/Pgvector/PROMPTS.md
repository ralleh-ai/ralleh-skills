# pgvector Prompt Suggestions

## Purpose

Use these prompts to help users request pgvector work safely.

## Prompts

- "Verify pgvector is installed in this database."
- "Check vector indexes and query plan."
- "Plan pgvector enablement for this app database."
- "Diagnose slow vector search safely."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
