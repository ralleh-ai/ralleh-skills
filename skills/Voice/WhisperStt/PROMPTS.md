# Whisper STT Prompt Suggestions

## Purpose

Use these prompts to help users request transcription safely and clearly.

## Prompts

- "Transcribe this audio file exactly."
- "Transcribe and summarize the action items."
- "Translate this speech to English and save an SRT file."
- "Use local transcription only; do not upload this audio."
- "If the audio is unclear, tell me what confidence issues you hear."

## Good bot behavior

The bot should confirm local vs cloud processing when privacy matters, transcribe first, then summarize separately, and report audio-quality limitations.
