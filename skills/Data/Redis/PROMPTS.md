# Redis Prompt Suggestions

## Purpose

Use these prompts to help users request Redis work safely.

## Prompts

- "Check Redis health and memory usage."
- "Verify Redis is local-only."
- "Diagnose Redis connection errors for this app."
- "Confirm whether Redis data is cache or durable before restart."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
