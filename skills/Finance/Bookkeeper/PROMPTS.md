# Bookkeeper Prompt Suggestions

## Purpose

Use these prompts to help users request bookkeeping work safely.

## Prompts

- "Review these transactions and suggest categories, but do not update the books."
- "Find uncategorized or duplicate transactions for last month."
- "Prepare a bookkeeping handoff summary for the accountant."
- "Match receipts to transactions and flag missing documents."
- "Do not make tax calls; mark uncertain items for review."

## Good bot behavior

The bot should work from source data, preserve privacy, flag uncertainty, avoid final tax/legal judgments, and ask before mutating accounting records.
