# LinkedIn Manager Skill

## What LinkedIn Manager is

LinkedIn Manager is the skill for managing LinkedIn profile, company page, content, comments, outreach drafts, and professional presence workflows.

It helps a bot draft, review, schedule, and analyze LinkedIn work while preventing accidental public posting or inappropriate outreach.

## What LinkedIn Manager does

- Drafts posts, comments, profile copy, company-page updates, and outreach messages.
- Reviews tone, audience fit, claims, and compliance risk.
- Checks published content or profile/page state when access is available.
- Requires approval before public posting, commenting, connecting, messaging, or profile changes.

## When a bot should use it

Use LinkedIn Manager when the user asks for LinkedIn posts, profile/page updates, content calendars, engagement review, outreach drafts, or analytics summaries.

Ask before any public or direct action.

## Files in this skill

- [`INSTALL.md`](INSTALL.md) — verify access method, account/page scope, and approval workflow.
- [`DOCTOR.md`](DOCTOR.md) — troubleshoot login, permission, page access, posting blockers, and analytics gaps.
- [`PROMPTS.md`](PROMPTS.md) — user-facing prompts for LinkedIn management.

## Golden operating rule

Draft freely, publish only after explicit approval. Never claim to represent the user or company beyond the approved voice and facts.
