# Skill Library Navigation

## Purpose

Teach bots how to use this repository without wandering or over-reading.

## Process

1. If given a bundle, read the bundle index first and follow its order.
2. If given a specific skill URL, read only that skill plus any linked prerequisite it names.
3. Prefer canonical category paths such as `skills/Memory/Engram/README.md`.
4. Use private overlays only after reading the public base skill.
5. Keep notes of which files were read when reporting completion.

## Path convention

```text
skills/<Family>/<Skill>/README.md       # full skill
skills/<Family>/<Skill-Name>.md         # compact instruction
bundles/<bundle-name>/...               # ordered reading list
templates/...                           # authoring templates
```
