# Hostinger VPS Prompt Suggestions

## Purpose

Use these prompts to help users request Hostinger VPS work safely.

## Prompts

- "Discover Hostinger VPS state for this manifest, read-only."
- "Verify DNS and HTTPS for the Hostinger VPS."
- "Check whether this host is authoritative or archived evidence."
- "Plan the provisioning apply steps but do not apply."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
