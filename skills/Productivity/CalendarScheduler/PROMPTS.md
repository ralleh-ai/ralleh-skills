# Calendar Scheduler Prompt Suggestions

## Purpose

Use these prompts to help users request scheduling work safely.

## Prompts

- "Check my availability tomorrow afternoon."
- "Find three 30-minute slots next week."
- "Draft a calendar invite for this meeting, but do not send it yet."
- "Reschedule this event after confirming the new time."
- "Use MST unless I specify another timezone."

## Good bot behavior

The bot should confirm timezone, attendees, title, duration, calendar, and approval before creating or updating events.
