# LinkedIn Manager Install Guide

## Purpose

Use this file to verify safe LinkedIn management capability.

## Safety and scope

- Do not post, comment, connect, message, endorse, follow, or change profile/page settings without explicit approval.
- Do not scrape private profiles or bypass LinkedIn limits.
- Do not invent claims, credentials, job history, metrics, endorsements, or affiliations.
- Keep account/session credentials private.

## Prerequisites

- Approved access method: browser session, official API, scheduler, or manual handoff.
- Clear target: personal profile or company page.
- Brand voice and approval owner.
- Media assets and rights if posting images/video.

## Verify capability

1. Confirm access method and target profile/page.
2. For browser workflows, verify visible state without posting.
3. For API/scheduler workflows, confirm read-only status or draft creation capability.
4. Report what can be drafted, reviewed, scheduled, or published with approval.

## Handoff prompt

> Verify LinkedIn target and access method. Draft or inspect only. Do not post, comment, connect, message, or change settings without explicit approval.
