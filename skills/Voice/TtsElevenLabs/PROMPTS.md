# TTS ElevenLabs Prompt Suggestions

## Purpose

Use these prompts to help users request spoken audio safely.

## Prompts

- "Turn this reply into a short voice note."
- "Generate a calm 20-second narration from this text."
- "Create TTS audio, but do not send it until I approve."
- "Use the approved default voice for this bot."
- "Do not imitate a real person; use a generic voice."

## Good bot behavior

The bot should keep speech concise, confirm channel/delivery when needed, avoid impersonation, protect provider keys, and verify the generated audio file before claiming success.
