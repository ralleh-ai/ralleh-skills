# Proposal Follow-Up Example

## Use When

The owner needs a post-proposal or post-demo follow-up draft.

## Input Checklist

- Prospect/account:
- Proposal/version:
- Last touch:
- Known objections:
- Approved terms/pricing:
- Desired next action:

## Output Shape

```text
Proposal Follow-Up — <Account>

Status:
- ...

Buyer concerns:
- ...

Draft follow-up:
Hi <Name>,
...

Approval needed before send:
- Yes — confirm recipient and final text.

Next action if no reply:
- ...
```

## Stop Conditions

Stop before including unapproved pricing, discount, delivery, legal, or implementation commitments.
