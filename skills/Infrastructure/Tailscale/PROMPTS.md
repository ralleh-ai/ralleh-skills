# Tailscale Prompt Suggestions

## Purpose

Use these prompts to help users request Tailscale work safely.

## Prompts

- "Check whether this VPS is on Tailscale and reachable."
- "Plan Tailscale tags/auth keys for this bot fleet without creating keys."
- "Verify private OpenClaw access over Tailscale."
- "Diagnose why this tailnet node is unreachable."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
