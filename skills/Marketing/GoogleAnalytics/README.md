# Google Analytics Skill

## What Google Analytics is

Google Analytics, usually GA4, is the analytics platform for website/app traffic, events, conversions, attribution, audiences, and reporting.

This skill guides bots through safe analytics reporting and instrumentation checks.

## What Google Analytics does

- Reads traffic, acquisition, engagement, event, and conversion reports.
- Checks whether expected events/conversions are being collected.
- Helps interpret trends and anomalies.
- Connects analytics questions to marketing/product decisions.

## When a bot should use it

Use Google Analytics when the user asks about traffic, conversions, campaigns, sources, events, attribution, funnels, pages, landing pages, or analytics setup.

Ask before changing property settings, audiences, conversions, links, data retention, or access permissions.

## Files in this skill

- [`INSTALL.md`](INSTALL.md) — verify GA4 access, property scope, date range, and read-only reports.
- [`DOCTOR.md`](DOCTOR.md) — troubleshoot missing data, wrong property, event issues, attribution confusion, and freshness delays.
- [`PROMPTS.md`](PROMPTS.md) — user-facing prompts for analytics reporting.

## Golden operating rule

Analytics answers must name property, date range, timezone, metric definitions, and known data limitations.
