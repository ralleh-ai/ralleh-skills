# SQLite Prompt Suggestions

## Purpose

Use these prompts to help users request SQLite work safely.

## Prompts

- "Check SQLite integrity and list tables."
- "Back up this SQLite database before migration."
- "Inspect schema and migration version without exposing rows."
- "Diagnose why this SQLite app is locked."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
