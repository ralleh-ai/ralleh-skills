# Docker Prompt Suggestions

## Purpose

Use these prompts to help users request Docker work safely.

## Prompts

- "Check Docker health and list running containers."
- "Review this Compose file for secrets, ports, volumes, and restart policy."
- "Plan a safe Compose deploy with rollback."
- "Diagnose why this container keeps restarting."

## Good bot behavior

The bot should inspect first, keep secrets private, ask before risky changes, preserve rollback where applicable, and verify the result before claiming success.
