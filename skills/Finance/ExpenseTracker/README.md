# Expense Tracker Skill

## What Expense Tracker is

Expense Tracker is the finance skill for capturing, organizing, categorizing, and reviewing expenses, receipts, reimbursements, and spend patterns.

It focuses on evidence-backed expense records and clear review queues.

## What Expense Tracker does

- Captures expense details from receipts, invoices, card exports, or user notes.
- Categorizes expenses using approved rules.
- Flags missing receipts, duplicates, policy exceptions, and reimbursement candidates.
- Produces spend summaries by category, vendor, project, or period.

## When a bot should use it

Use Expense Tracker when the user asks to log expenses, review spend, match receipts, prepare reimbursement packets, or summarize expenses.

Ask before submitting reimbursements, changing accounting records, or sending financial data externally.

## Files in this skill

- [`INSTALL.md`](INSTALL.md) — verify expense sources, receipt handling, and categorization rules.
- [`DOCTOR.md`](DOCTOR.md) — troubleshoot missing receipts, duplicates, unclear categories, or export issues.
- [`PROMPTS.md`](PROMPTS.md) — user-facing prompts for expense workflows.

## Golden operating rule

Every expense entry should trace back to evidence: receipt, invoice, transaction row, or explicit user note.
