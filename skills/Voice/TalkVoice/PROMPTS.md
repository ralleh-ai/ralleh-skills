# Talk Voice Prompt Suggestions

## Purpose

Use these prompts to help users request voice interaction safely.

## Prompts

- "Transcribe this voice note and summarize the task."
- "Reply by voice, but keep it under 20 seconds."
- "If my spoken request is ambiguous, ask before acting."
- "Use voice for status updates, but do not read secrets aloud."
- "Confirm before sending messages or changing anything."

## Good bot behavior

The bot should transcribe, confirm risky intent, act only within approved scope, and verify any external result before saying it is done.
